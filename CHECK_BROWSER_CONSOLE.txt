COMMON ERRORS:
==============

Error: "The query requires an index"
Solution: 
  - Check Firebase Console indexes
  - Make sure all are GREEN (Enabled)
  - Wait 5 more minutes if YELLOW (Building)

Error: "Permission denied"
Solution:
  - Make sure you're logged in
  - Check that you're logged in as admin
  - Run: cd functions && npm run create-admin

Error: "Query returned 0 documents"
Solution:
  - Check security rules are allowing read
  - Check that you're authenticated
  - Try logging out and back in
